Comment Power Apps s’intègre à l’ensemble de votre environnement Microsoft ?

AVEC MICROSOFT 365

Microsoft Power Apps est disponible via Microsoft 365, avec des connecteurs standards qui vont permettre de mettre en œuvre un grand nombre d’applications.

AVEC LA POWER PLATFORM

Développez des solutions professionnelles end-to-end en connectant Power Apps avec l’ensemble des applications de Microsoft Power Platform (Power Automate, Power BI, Power Virtual Agents, Dataverse).

AVEC LE CLOUD AZURE

Microsoft Azure vous permet d’étendre les applications « low code » développées avec Power Apps afin de créer des solutions évolutives. Gérez votre développement à l’aide d’Azure DevOps et tirez parti des services d’IA et des données sur Azure.
